# air-compressor-troubleshooting
Rotary Screw Troubleshooting
This repository contains an original, structured troubleshooting logic system for
oil-injected rotary screw air compressors.

## Purpose
- Guide users through safe, step-by-step diagnostics
- Identify common operational issues
- Escalate advanced problems to professional service

## Scope
This logic is intended for:
- Web applications
- Mobile apps
- Technician support tools
- Customer-facing troubleshooting guides

## Safety
Always disconnect electrical power and relieve system pressure before inspection.
This logic does not replace qualified service personnel.

## File Structure
- `troubleshooting.logic.json` — Core decision-tree logic
- `README.md` — Project documentation
